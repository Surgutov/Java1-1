KeyValidator_Valid
Провека валидных ключей при помощи программы KeyValidator

Открыть страницу https://github.com/netology-code/javaqa-homeworks/blob/master/intro/artifacts/KeyValidator.class;
Загрузить файл KeyValidator.class нажав кнопку "Download";
Перенести сохраненный файл в папку c:\Users\User;
Запустить командную строку на рабочей станции сочетанием клавиш Win+R;
В открывшемся окне ввести команду cmd;
Проверить версию установленной версии Java при помощи команды java - version;
Запустить KeyValidator при помощи команды java KeyValidator и ввести валидные ключи единым списком (копируя из исходного ТЗ и вставляя в командную строку), разделяя номера пробелами или по одному;

Ожидаемый результат:
Все валидные ключи из технического задания прошли проверку: Result for <key number>: OK
